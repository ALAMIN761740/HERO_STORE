# Mission Restart

React + Vite + Tailwind CSS + daisyUI scaffold

Setup:

```bash
# 1. install deps
npm install

# 2. run dev server
npm run dev

# 3. build for production
npm run build

# 4. preview production build
npm run preview
```

Notes:
- Tailwind and daisyUI are preconfigured in `tailwind.config.cjs`.
- If you prefer `yarn` or `pnpm`, replace `npm install` accordingly.

Happy hacking!"
